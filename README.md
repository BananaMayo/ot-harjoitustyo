# MazeGame

MazeGame on pygamella toteutettu hasuka pieni peli. Pelissä on labyrintti jossa tavoitteena on päästä ulos, löytämällä uloskäynti.

## Python-versiosta
Sovellus on toiminnallisuudeltaan testattu Python-versiolla `3.8`
